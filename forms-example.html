Complete working example with Newsletter, Inquiry, Order, and Feedback forms

Setup Instructions

⚠️ IMPORTANT: Before using these forms, you must set your Manus backend URL!

Step 1: Update Backend URL

Open browser console (F12) and run:

setManusBacendUrl('https://your-manus-backend.com/api/trpc');

Step 2: Verify Configuration

Run this to check your setup:

logApiConfig();

Step 3: Test Forms

Fill out any form below and submit. Check browser console (F12) for logs and responses.
